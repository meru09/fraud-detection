# 💳 Credit Card Fraud Detection

A machine learning project that explores, analyzes, and builds models to detect fraudulent credit card transactions.

---

## 📌 Project Overview

Credit card fraud is a major financial problem that causes billions of dollars in losses every year. The objective of this project is to analyze transaction patterns and build machine learning models capable of distinguishing fraudulent transactions from legitimate ones.

---

## 🎯 Objectives

- Understand the dataset through exploratory data analysis (EDA)
- Investigate fraud patterns
- Handle severe class imbalance
- Build and compare multiple machine learning models
- Evaluate model performance using appropriate metrics
- Create a reproducible machine learning pipeline

## 📊 Dataset

**Dataset:** Credit Card Fraud Detection Dataset

The dataset contains anonymized credit card transactions made by European cardholders.

Features:
- Time
- Amount
- V1 – V28 (PCA-transformed features)
- Class (Target)

Target:
- 0 → Legitimate transaction
- 1 → Fraudulent transaction
